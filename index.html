Structure tag

Tag Description
!doctype declaration Tells browser HTML 5 is being used.
html start and end of the html document.
head Contains the meta data of the website
title Contains the title of the website
body Contains the content of the website
link Used to link to external files like stylesheet, fonts

Basics tag

Tag Description
h1 to h6 represent the six level of heading
p paragraph tag
div it groups different unrelated tags.
span it groups different unrelated elements i.e inline
em it emphasizes the text
strong it highlights the text
br breaks the line. don't need closing
ul creates unordered list
ol creates ordered lists
li creates list items for ul and ol
a creates hyperlink
img add image to our site
button creates button
video adds video elements to our page.

Table tag

Tag Description
table it contains the tabular data
tr it creates row in our table.
td to add data we use td tag
th table needs heading which is adds by this tag.
thead sectioned off the heading part of the table.
tbody sectioned off the body part of the table
tfoot sectioned off the footer part of the table

Semantic tag

Tag Description
header represent the introductory content of the page
nav represent the navigational item like menu, table of content
main contains the dominant contant of the page
aside represent the content aside from the main content
section represent the section of the page
article represent the article of the page, like blog, comments
figure it encapsulate media such as image, llustration, diagram
figcaption represent the caption of the figure
Audio it encapsulate audio content on the page
video it contains the video elements of the page.
footer represent the conclusion of the page
